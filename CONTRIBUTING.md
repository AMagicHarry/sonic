# How to get started with contributing

- First of all, fork and clone this repo
- Install Rust and Cargo
- Install npm

## Build

From repo root run

```sh
cargo build
```

## Start Soniq from source

From repo root run

```sh
cargo run
```

## Unit test

From repo root run

```sh
cargo test
```

## Integration test

From `<repo root>/tests/integration/scripts/` run

```sh
./run.sh
```
